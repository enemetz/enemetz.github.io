Websight for iOS

With no way to easily get information from my PC to my phone without emailing myself, I Designed and built a computer vision iOS app to help with this. Websight lets a user scan data from the camera, then websight will detect the type of data, and then prompt the user with shortcuts based on the data type. Websight also allows a user to extract text from images, Enabling a user to select an image of theirs, and get the text back to copy to their clipboard.
